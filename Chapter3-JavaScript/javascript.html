第三章 JavaScript

JavaScript

JavaScript一种直译式脚本语言，是一种动态类型、弱类型、基于原型的语言，内置支持类型。它的解释器被称为JavaScript引擎，为浏览器的一部分，广泛用于客户端的脚本语言，最早是在HTML网页上使用，用来给web网页增加动态功能

任何可以使用 JavaScript 来编写的应用，最终会由 JavaScript 编写
