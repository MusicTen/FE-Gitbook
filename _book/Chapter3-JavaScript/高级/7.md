# 递归

函数中自己调用自己，，递归一定要有结束条件

> 化归思想，将一个问题由难化易，由繁化简，由复杂化简单的过程称为化归

### 案例1：斐波那契数列

```javascript
function getFib(x) {
    if (x == 1 || x == 2) {
        return 1;
    }
    return getFib(x - 1) + getFib(x - 2);
}
console.log(getFib(12));
```

### 案例2：阶乘

```javascript
function factorial(n){
    if (n == 1 || n == 0) {
        return 1;
    }
    return n * factorial(n-1);
}
console.log(factorial(10));
```
